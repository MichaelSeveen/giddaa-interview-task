# Giddaa Interview Task (Tax Oga)

This is the tax oga website for Giddaa, built using Next.js. Here is the live link: [giddaa-interview-task](https://giddaa-interview-task.vercel.app)

# Getting Started

Follow these steps to run the project locally:

1. Clone the Repository:
   You can either download the ZIP file of the repository or clone it using Git:

   ```bash
   https://github.com/MichaelSeveen/giddaa-interview-task.git
   ```

2. Navigate to the Project Directory:
   Change into the project directory:

   ```bash
   cd <project-directory> i.e. cd giddaa-interview-task
   ```

3. Install Dependencies:
   Use bun to install the necessary packages:

   ```bash
   bun install
   ```

4. Run the Development Server:
   Start the development server with:

   ```bash
   bun run dev
   ```

5. Open in Browser:
   Open [localhost:3000](http://localhost:3000) in your browser to view the homepage and [localhost:3000/tax-calculator](http://localhost:3000/tax-calculator) to view tax calculator page.
